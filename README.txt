Reglas de uso:

-cada palabra debe estar separada por un espacio


ejemplos de uso:

para entero i = 0 ; i <= 100 ; i ++ finC

salida esperada :

[LEXEMA: para] ---- [TIPO: Ciclo inicio-fin]
[LEXEMA: entero] ---- [TIPO: Declaracion]
[LEXEMA: i] ---- [TIPO: Nombre variable]
[LEXEMA: =] ---- [TIPO: Asignacion]
[LEXEMA: 0] ---- [TIPO: valor numerico]
[LEXEMA: ;] ---- [TIPO: Separador de linea]
[LEXEMA: <=] ---- [TIPO: Operador Logico]
[LEXEMA: 100] ---- [TIPO: valor numerico]
[LEXEMA: ++] ---- [TIPO: Operacion Aritmetica]
[LEXEMA: fin] ---- [TIPO: Nombre variable]
